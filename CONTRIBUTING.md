# Contributing to the Modern Portfolio Template

First off, thank you for considering contributing to this project! Your help is appreciated.

## How Can I Contribute?

### Reporting Bugs
- **Ensure the bug was not already reported** by searching on GitHub under [Issues](https://github.com/mdnoyon9758/modern-portfolio-template/issues).
- If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/mdnoyon9758/modern-portfolio-template/issues/new).

### Suggesting Enhancements
- Open an issue and provide as much detail as possible about your suggestion.

### Pull Requests
- Fork the repository and create your branch from `main`.
- Make sure your code lints and passes tests.
- Issue that pull request!

## Styleguides

### Git Commit Messages
- Use the present tense ("Add feature" not "Added feature").
- Use the imperative mood ("Move cursor to..." not "Moves cursor to...").
- Limit the first line to 72 characters or less.
- Reference issues and pull requests liberally after the first line.

### Code Style
- For frontend, we use Prettier and ESLint. Please run `npm run format` and `npm run lint` before committing.
- For backend, we use Black and Ruff. Please run `black .` and `ruff check .` before committing.
